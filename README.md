# NCT - Neo College Technology Website

Website Sistem Informasi & Berita Neo College Technology (NCT) adalah platform berbasis web yang dibangun untuk menyediakan informasi, berita kampus, serta manajemen konten yang terintegrasi melalui sistem admin panel.

Website ini sudah di-deploy dan dapat diakses secara online melalui Railway.

## 🔗 Live Demo: https://nct.up.railway.app/

# ✨ Fitur Utama
## 🌐 Public Website (Landing Page)
- Halaman landing page modern dan responsif
- Tampilan informasi utama tentang NCT
- Akses cepat ke berita terbaru
- Navigasi antar halaman yang sederhana dan user-friendly
## 📰 News / Berita
- Menampilkan berita terbaru dari NCT
- Detail artikel berita dengan tampilan rapi
- Sistem kategori berita (jika tersedia di backend)
- Update konten secara dinamis melalui admin
## 🏢 Department (Departemen)
- Informasi struktur atau divisi dalam NCT
- Data dapat dikelola melalui admin panel
- Mendukung pengembangan struktur organisasi kampus
## 🔐 Admin Panel
- Login admin untuk manajemen sistem
- Dashboard admin untuk kontrol data
- CRUD (Create, Read, Update, Delete) untuk User dan Berita
- Akses cepat ke halaman website utama dari admin
